# Problem-solving-dsa-python

This repository contains solutions to various data structure and algorithm problems implemented in Python. The problems are organized into categories based on their difficulty and type.

## Directory Structure

- **Arrays-Easy/**: Contains solutions to easy-level problems related to arrays.
  - **Q1-Largest Element in an Array/**: Solution for finding the largest element in an array.
  - **Q2-Second Largest Element in an Array/**: Solution for finding the second largest element in an array.
  - **Q3-Find the First Unique Element in an Array/**: Solution for finding the first unique element in an array.
  - **Q4-Check if an Array is Sorted/**: Solution for checking if an array is sorted.
  - **Q5- Remove Duplicates in-place from Sorted Array/**: Solution to remove duplicates inplace from sorted array.
  - **So On ...**

- **Sorting-Methods/**: Contains various sorting algorithms.

## Problem Statements

Each problem folder contains:
- `Question.md`: A markdown file describing the problem statement, examples, and constraints.
- `Answer.py`: A Python file containing the solution to the problem.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/problem-solving-dsa-python.git
   ```
2. Navigate to the desired problem directory:
   ```bash
   cd Arrays-Easy/Q1-Largest Element in an Array
   ```
3. Run the solution:
   ```bash
   python Answer.py
   ```
