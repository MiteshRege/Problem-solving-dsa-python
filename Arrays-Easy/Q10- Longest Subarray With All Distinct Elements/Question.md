# Longest Sub-array with All Distinct Elements 

## Problem Statement:
- Given an array, return the length of the longest contiguous subarray where all elements are unique.

## Examples:
- Input: [1,2,1,3,4,3,5]
- Output: 4   (subarray = [1,3,4,3]? No — repeated!
              correct = [1,2,1,3]? No — repeated!
              correct = [2,1,3,4] ✅ length 4)

- Input: [5,5,5,5]
- Output: 1

- Input: [1,2,3,4,5]
- Output: 5
